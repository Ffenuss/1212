Мобильное приложение (PWA) — калькулятор зарплаты
=================================================

Что внутри:
- index.html — основное приложение.
- manifest.webmanifest — манифест для установки на телефон.
- service-worker.js — офлайн-режим для PWA.
- icons/ — иконки приложения.

Как использовать:
1. Открой index.html в браузере — приложение сразу работает как мобильная веб-версия.
2. Чтобы установить как приложение на телефон, файлы нужно открыть через HTTPS (например GitHub Pages, Netlify, Vercel или любой хостинг).
3. После открытия по HTTPS в поддерживаемом браузере появится кнопка «Установить».
